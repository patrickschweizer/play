## Setup environment
- install node.js
- start node command line
- create a project folder and jump into it 

- open http://yeoman.io/ and follow setup description
  - npm install -g yo    (also installs grunt and bower)
  - npm install -g generator-webapp    (install webapp generator)
  - yo webapp          (scaffold out a skeleton web app project)
  --> hint:  remove compass referencies, if compass is not isntalled

- grunt server   
- start editing...  


## Setup GIT
- cd working dir
- git init  (creates local repo)
- git status
- git add .  (add files to repo)

- create new repo at githup.com
- git remote add origin https://github.com/patrickschweizer/play.git
- git push -u origin master
- git pull
